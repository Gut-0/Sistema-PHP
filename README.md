## Como rodar a aplicação
Para rodar o projeto, você deve fazer o seguinte:

Clonar e abrir o projeto
````
git clone git@gitlab.com:Gut0/crud-com-angular.git
cd moments
code .
````
Instale as dependências
````
npm i
````
Execute o projeto
````
ng serve
````
O projeto irá rodar em [http://localhost:4200](http://localhost:4200)
